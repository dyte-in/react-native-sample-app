## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.


### Installation

1. Clone the repo

```sh
https://github.com/dyte-in/react-native-sample
```

2. Install NPM packages

```sh
npm install
```

3. Pod Install 

```
cd ios && pod install
```
